# 📔 ツヅク日記

毎日の気分と出来事を記録する日記アプリ。

気分は5段階の絵文字でワンタップ記録。カレンダーで俯瞰し、統計で自分のパターンを発見できます。

## 機能

- 😄 気分5段階記録 (最高・良い・普通・悪い・最悪)
- ✍️ フリー日記 (自動保存)
- 🏷️ 活動タグ + カスタムタグ
- 🔥 連続記録ストリーク
- 📅 月間カレンダー (気分を色分け表示)
- 📊 統計 (気分推移・気分割合・タグランキング)
- 🌙 ダーク/ライトモード切替

## 使い方

`index.html` をブラウザで開くだけで動きます。サーバー不要。

## 技術

HTML / CSS / JavaScript / Canvas API / LocalStorage
